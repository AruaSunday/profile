Introduction To Web Programming(CSC121)

A: COURSE AIM AND OBJECTIVES

AIM

This course is intended to equip the students with the basic skills, knowledge and tools required to becoming proficient in website development.

OBJECTIVES

The objective of this course is to teach the student to be able to:
  1. Defind and discuss the evolution of the WWW and explain the meaning of Internet, Intranet and Extranet.
  2. List the devices used to access the internet and explain the various means of accessing the internet
  3. Differentiate between static and dynamic web pages
  4. Describe the term "Computer Network"
  5. Use web browsers to display HTML codes
  6. Use web browsers to display XHTML codes
  7. etc

B: LEARNING OUTCOMES

Upon the completion of the course, the student will be able to:
  • Understand the basic concept, features and services of the internet
  • Write a program interface for both HTML and XML documents
  • Understand and use various HTML tags for web development
  • Understand the relevance of XML in web development
  • etc

C: COURSE CONTENT

Web programming basics. Fundamentals of developing web pages using a comprehensive web development life cycle. Introduction to the Internet. Internet services and protocols. Hypertext Transfer Protocol (HTTP). Transport control protocol/Internet protocol (TCP/IP). Internet Hardware Requirements. Network Model. Web Application Development. Client-Server environment. Browsers and Web Servers.
