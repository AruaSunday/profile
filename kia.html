Course Content

Web programming basics. Fundamentals of developing web pages using a comprehensive web development life cycle. Introduction to the Internet. Internet services and protocols. Hypertext Transfer Protocol (HTTP). Transport control protocol/Internet protocol (TCP/IP). Internet Hardware Requirements. Network Model. Web Application Development.

Ojectives

The objective of this course is to teach the student to be able to:
  1. Defind and discuss the evolution of the WWW and explain the meaning of Internet, Intranet and Extranet.
  2. List the devices used to access the internet and explain the various means of accessing the internet
  3. Differentiate between static and dynamic web pages
  4. Describe the term "Computer Network"
  5. Use we browsers to display HTML codes
